# XCBuild

Run `xcodebuild` with arguments read from a configuration file in the local directory: `.xcbuild.yml`.
Format the output with `xcpretty`, and you have a modern and pretty `xcodebuild`.

## Installation

    $ gem install xcbuild

## Usage

    $ xcbuild
